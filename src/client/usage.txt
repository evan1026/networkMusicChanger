Usage:
  networkMusicCLI (play|pause|next|prev|get) [--url=URL] [--port=PORT]
  networkMusicCLI --version | --help

Options:
  -u URL, --url=URL     The url of the server to connect to [default: localhost]
  -p PORT, --port=PORT  The host port [default: 41900]
  -h, --help            Show this screen
  -v, --version         Display version and exit.
